# CancerCare Platform - Level 2 Student Project

## Project Overview
This is a crowdfunding website for cancer patients in Yaoundé, Cameroon. It was created as a student project to help patients raise money for their medical treatment.

## What Makes This a Level 2 Student Project

### 1. **Clean HTML Structure**
- Well-organized HTML5 pages with semantic structure
- Consistent navigation menu across all pages
- Properly structured forms with labels and inputs
- No complex frameworks - just vanilla HTML/CSS/JS

### 2. **Simple but Professional CSS**
- Clean, modern design that looks professional
- Basic CSS properties (colors, padding, margins, borders)
- Simple grid layout for cards
- Basic transitions and hover effects
- Readable and well-commented code

### 3. **Straightforward JavaScript**
- Simple array-based data storage
- Basic DOM manipulation
- Form validation using simple if/else statements
- Functions that are easy to understand
- No complex algorithms or advanced patterns

### 4. **Realistic Student Features**
- 5 patient campaigns (2 children, 3 adults)
- Mobile money donation system (MTN, Orange)
- User registration and login
- Hospital verification system
- Health events listing

### 5. **Appropriate Complexity**
- Professional-looking but achievable for students
- Shows good understanding of web fundamentals
- Well-organized code structure
- Good balance between simplicity and functionality
- Demonstrates practical application of HTML/CSS/JS
