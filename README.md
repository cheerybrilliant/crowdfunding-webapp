## How to Use

1. **Open the Project**: Simply open `index.html` in your web browser to start
2. **Navigate**: Use the navigation bar to move between pages
3. **Study the Code**: Each file is well-commented and easy to understand

## Pages Description

### 1. Home Page (index.html)
- Hero section with platform introduction
- Three information cards explaining features
- Links to create account and donate

### 2. Create Account (create-account.html)
- Form to register as donor or patient
- Email and password validation
- Redirects patients to verification page

### 3. Login (login.html)
- Simple login form
- Demo credentials: any valid email and password (6+ characters)
- Links to forgot password and hospital login

### 4. Forgot Password (forgot-password.html)
- Email input for password recovery
- Simulates password reset process

### 5. Campaigns (campaigns.html)
- Displays all patient fundraising campaigns
- Shows progress bars and funding status
- Links to view details and donate

### 6. Campaign Details (campaign-details.html)
- Full patient story
- Detailed funding information
- Donate button

### 7. Donate (donate.html)
- One-time donation form
- Mobile money payment options (MTN, Orange)
- Quick amount buttons

### 8. Verification (verification.html)
- Patient document upload
- Profile picture and video upload
- For patients only

### 9. Events (events.html)
- Calendar view of upcoming events
- List of health seminars and gatherings
- Registration for events

### 10. Partner Hospitals (partner-hospitals.html)
- List of 5 partner hospitals in YaoundÃ©
- Hospital contact information and services
- Verification process information
- Partnership opportunities

### 11. Hospital Login (hospital-login.html)
- Admin login for hospitals
- Demo credentials: HOSP001 / hospital123

### 12. Hospital Dashboard (hospital-dashboard.html)
- Patient verification management
- Campaign overview
- Event creation
- Statistics dashboard

### 13. About Us (about.html)
- Team introduction
- Mission statement
- Transparency commitment
- Partner hospitals information

## Key Features

### CSS (styles.css)
- **Responsive Design**: Works on different screen sizes
- **Modern UI**: Clean and professional look
- **Reusable Classes**: Easy to maintain and extend
- **Color Scheme**: Professional blue and green colors

### JavaScript (script.js)
- **Form Validation**: Email and password validation
- **Local Storage**: Saves user data in browser
- **Sample Data**: Pre-loaded campaigns and events
- **Helper Functions**: Currency formatting, progress bars
