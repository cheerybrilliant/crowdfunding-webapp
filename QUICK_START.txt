================================================================================
COMMON TASKS
================================================================================

CHANGE COLORS:
  1. Open styles.css
  2. Find color codes (e.g., #3498db)
  3. Replace with new colors
  4. Save and refresh browser

CHANGE TEXT:
  1. Open any .html file
  2. Find the text you want to change
  3. Edit it
  4. Save and refresh browser

ADD A NEW CAMPAIGN:
  1. Open script.js
  2. Find "sampleCampaigns" array
  3. Copy an existing campaign object
  4. Modify the details
  5. Save and refresh browser

ADD A NEW EVENT:
  1. Open script.js
  2. Find "sampleEvents" array
  3. Copy an existing event object
  4. Modify the details
  5. Save and refresh browser
